스크립트 적용 방식

  • 인라인 스크립트: 해당 태크에 직접 스크립트를 기입하는 방식
  • 내부 스크립트: 해당 문서에 script 태그를 깅비하여 스크립팅하는 방식
  • 외부 스크립트: 별도의 .js 파일을 작성하여 여러 문서에서 script 태그를 삽입하는 방식

스크립트 출력 방향

  • 문서에 출력: document.write()
  • 요소에 출력: document.getElementById().innerHTML
  • 콘솔에 출력: console.log()

메세지 창

  • 알림창: alert("메세지")
  • 확인창: confirm("메세지")
  • 입력창: prompt("메세지")
